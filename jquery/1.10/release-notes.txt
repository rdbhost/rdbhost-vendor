		<h1 class="entry-title">jQuery 1.10.2 and 2.0.3 Released</h1>
				<div class="entry-posted">
			<span class="sep">Posted on </span><a href="http://blog.jquery.com/2013/07/03/jquery-1-10-2-and-2-0-3-released/" title="8:55 am" rel="bookmark"><time class="entry-date" datetime="2013-07-03T08:55:03+00:00" pubdate>July 3, 2013</time></a><span class="by-author"> <span class="sep"> by </span> <span class="author vcard"><a class="url fn n" href="http://blog.jquery.com/author/dmethvin/" title="View all posts by dmethvin" rel="author">dmethvin</a></span></span>		</div>
				<hr>

		
<p>It&#8217;s nearly Independence Day here in the USA, so we&#8217;re delivering something fresh off the grill: jQuery 1.10.2 and 2.0.3. These releases fix a few pesky bugs that have been reported over the past month, but the list is refreshingly small. Since some of the bugs spanned both the 1.x and 2.x branches we&#8217;re releasing new versions of both to keep them in sync. </p>
<p>You can get the latest files from the <a href="http://jquery.com/download/">jQuery download page</a>, including sourcemap files and links to helpful tools such as jQuery Migrate. If you&#8217;re upgrading from a version of jQuery before 1.9, please do read through that page carefully to make your migration as pain-free as possible. Remember that it may take a few days for the CDNs at Google, Microsoft, and CDNJS to respond to the rocket&#8217;s red glare and post the latest versions. In the meantime, use the copy on the jQuery CDN.</p>
<p>We&#8217;re pretty optimistic that these latest bug-fix releases should be free of surprises. If you drop the new files into your site and see fireworks, please do your patriotic duty and report a bug with a test case (preferably using <a href="http://jsfiddle.net">jsFiddle</a>) at our <a href="http://bugs.jquery.com">bug tracker</a>.</p>
<p>These releases wouldn&#8217;t have happened without the contributions of Jason Bedard, Jason Merino, JÃ¶rn Zaefferer, Micha&#322; Go&#322;&#281;biowski, Nguyen Phuc Lam, Oleg Gaidarenko, Richard Gibson, Rick Waldron, Terry Jones, and Timmy Willison.</p>
